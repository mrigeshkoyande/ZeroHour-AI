## 🚀 Core Features & Autonomous Modules

### 1. 🧠 Autonomous Command Center (`/dashboard`)
* **Real-Time Risk Gauge:** Continuously calculates your daily workspace collision probability using SLA risk scoring.
* **Focus Velocity Analytics:** Tracks productivity efficiency trends across weekly pulse and monthly horizons.
* **Deadline Risk Radar:** Highlights upcoming deliverables sorted by critical, medium, and low bottleneck severity.
* **Interactive AI Suggestions Feed:** One-click automated mitigation actions generated by the system's background worker.

### 2. ⚡ Deadline Risk Engine (`/risk`)
* **Live Risk Modulation Simulator:** Interactive sliders allowing you to test how reducing meeting durations by 25–50% or enabling autonomous delegation impacts your SLA safety score.
* **48-Hour Horizon Collision Matrix:** Deep breakdown of conflicting sprints, investor reviews, and product launches with instant AI relocation strategies.

### 3. 🛡️ SOS Recovery Shield (`/sos`)
* **1-Click Emergency Protocol:** Designed for intense burnout or sudden critical emergencies.
* **Action Automation:** Automatically freezes non-essential Slack/email alerts, places protective placeholders over calendar deep-work slots, and drafts professional delay explanations for stakeholders.

### 4. 🧬 Life Digital Twin (`/twin`)
* **Circadian Peak Window Mapping:** Analyzes neurological efficiency windows to automatically schedule heavy architectural decisions during peak cognitive hours (e.g., 09:30 AM – 11:45 AM).
* **Fatigue & Burnout Runway:** Monitors back-to-back meeting drag and calculates safe operational runway before fatigue sets in.

### 5. 📅 AI Defragmenting Calendar (`/calendar`)
* **1-Click Auto-Resolve:** Scans for overlapping events and instantly defragments the calendar without impacting meeting SLA commitments.
* **Block Filtering:** Filter between *Deep Work Shields*, *Stakeholder Syncs*, and *Cognitive Decompress* windows.

### 6. 🔮 AI Future Horizon Planner (`/planner`)
* **High-Velocity Milestone Injection:** Create new deliverables where Gemini automatically calculates and pads a **15% safety buffer** to protect against timeline slippage.
* **Automated Risk Assessment:** Real-time feedback on team bandwidth saturation prior to committing to deadlines.

### 7. 💬 Gemini 3.1 Pro Live Copilot & Universal Search (`⌘K`)
* **Floating Command Palette:** Instant access from anywhere in the OS to search tasks, meetings, people, or ask Gemini for scheduling guidance.
* **Dedicated Executive Chat (`/chat`):** Strategic sounding board with context-aware prompts (*Schedule Audit*, *Draft Delay Explanations*, *Prep Executive Summaries*).

### 8. 📊 Executive Analytics & Autonomy Settings (`/analytics` & `/settings`)
* **Productivity Velocity Tracking:** Deep visualization of AI auto-fixes executed and hours saved.
* **Modulable Autonomy Levels:** Switch between *Conservative* (Ask before acting), *Balanced* (Auto-reschedule low priority), and *Founder Mode* (Full autonomous execution).

## 🎨 Design Philosophy & UI Architecture

ZeroHour AI strictly follows high-end minimalist design standards:
* **Light Mode Luxury:** Clean `#FFFFFF` card foundations set against a soft `#F8F9FA` workspace canvas.
* **Curated Color Palette:** Emerald Green (`#10B981`) primary action tokens paired with vibrant Orange (`#FB923C`) and Amber (`#F59E0B`) accent alerts.
* **Responsive Bento Grids:** Precision 12-column grid architecture engineered to scale immaculately across mobile (`col-span-12`), tablets (`md:col-span-6`), laptops, and 4K ultra-wide monitors (`xl:col-span-4`).
* **Micro-Typography:** Powered by Google Fonts **Sora** (Headlines & Numerals) and **Inter** (Data tables & body copy).

## ⚡ Quickstart Guide

### Prerequisites
* **Node.js** `v18.0.0` or higher
* **npm** `v9.0.0` or higher

### 1. Clone & Install Dependencies
```bash
# Navigate into the project directory
cd ZeroHour-AI

# Install exact dependencies
npm install
```

### 2. Launch Local Development Server
```bash
npm run dev
```
The application will start immediately with instant hot-reloading at:
👉 **http://localhost:5174/** *(or `http://localhost:5173/`)*

### 3. Verify Production Build
To verify type safety and compile the minified production bundle:
```bash
npm run build
```
*(Build executes clean TypeScript checking `tsc -b` and outputs optimized assets into `/dist` in under 300ms).*
